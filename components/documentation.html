I like to keep the header simple. No need to overcomplicate this, especially if it is supposed to be versatile enough to work for multiple projects.

Navigation

I like my navigations to cover the entire width of the page. This is veratile enough to be a header by itself or sit underneath the header. I like to keep the color changes of the nav links to be subtle enough for the user to tell when they've hovered over a specific link.

Typography

Default Typefaces Used:

  • Brandon Grotesque
  • Rooney Web

For my default typefaces I wanted to create a pairing that would work relatively universally. I settled on Brandon Grotesque for Headers, Navigation, and Short Asides. I felt this typeface performed well in each of these scenarios. I chose Rooney Web because it is a serif font has just enough style so that it will work well for body copy.

Documentation Typefaces Used:

  • Chaparral Pro
  • Garamond Pro

For my documentation typefaces I wanted to create a pairing that would be a little more sterile and provide a clean, manual style feel. I chose Chaparral Pro for headings and Proxima Nova for body copy, because I felt they accomplished this goal.
